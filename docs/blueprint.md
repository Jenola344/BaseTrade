# **App Name**: BaseTrade

## Core Features:

- Market Data Feed: Real-time display of asset prices and market data.
- Portfolio Management: User interface for viewing and managing derivatives portfolios.
- Order Execution: Order placement for perpetual futures, options, and synthetic assets.
- AI Trading Assistance: AI-powered tool for generating basic trading signals and insights.
- Real-time Analytics: Display options Greeks, trading volume, and other relevant analytics.
- PWA Support: Progressive Web App (PWA) features for cross-platform access.

## Style Guidelines:

- Primary color: Royal blue (#4169E1) to convey trust, security, and professionalism.
- Background color: Light gray (#EEEEEE), near-white, providing a clean, neutral backdrop.
- Accent color: Sky blue (#87CEEB), for highlighting interactive elements and key information.
- Body and headline font: 'Inter' (sans-serif) for a modern, clean, and highly readable interface. Its neutral appearance ensures it won't compete with data-heavy layouts.
- Use minimalist line icons to represent different asset classes, portfolio metrics, and trading functions.
- Prioritize a clear, intuitive layout with well-defined sections for market data, portfolio overview, and order entry.
- Subtle transitions and animations to provide feedback on user interactions.
