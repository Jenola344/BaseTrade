# BaseTrade

A perpetuals trading platform built with React and Firebase, featuring an integrated AI trading assistant for market analysis and signal generation.

## What It Does

BaseTrade lets you trade perpetual contracts while getting real-time insights from an AI assistant. The assistant analyzes market data, identifies patterns, and generates trading signals to help inform your decisions.

**First Time Setup:**
1. Create an account using email or social login
2. Connect your crypto wallet
3. Deposit funds to your trading account
4. Turn on the AI assistant in settings

**Trading:**
- View live market data and price charts
- Review AI-generated signals in the assistant panel
- Set your risk parameters (leverage, stop loss, take profit)
- Execute trades directly from the interface
- Monitor open positions in the portfolio view

**AI Assistant Features:**
- Market trend analysis based on technical indicators
- Risk scoring for potential trades (low/medium/high)
- Price movement predictions with confidence levels
- Pattern recognition alerts
- Portfolio rebalancing suggestions

**AI Components:**
- Custom ML models for signal generation
- Technical analysis algorithms
- Real-time data processing pipeline
